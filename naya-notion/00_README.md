# 📋 Instructions d'import — Notion "Naya Demo"

Bienvenue dans le kit Notion de Naya. Ces 10 pages sont prêtes à publier.
Suis les étapes ci-dessous, ça prend moins de 10 minutes.

---

## Comment importer dans Notion

### Étape 1 — Importer les fichiers

1. Ouvre Notion sur desktop (l'import ne fonctionne pas sur mobile)
2. Dans la sidebar, clique sur **"+ Nouvelle page"**
3. Nomme cette page **"Naya Demo"**
4. Pour chaque fichier `.md` (de `01_accueil.md` à `10_prochaines-etapes.md`) :
   - Fais glisser le fichier directement dans la sidebar, **sous "Naya Demo"**
   - Notion le convertit automatiquement en page

### Étape 2 — Réorganiser les pages

1. Une fois tous les fichiers importés, ouvre la page **"Naya Demo"**
2. Glisse chaque sous-page dans l'ordre numéroté (01 → 10)
3. Tu peux renommer les pages en supprimant le préfixe numérique si tu préfères

**Ordre recommandé dans la sidebar :**

- 🌿 Accueil
- 🎬 Démo vidéo
- ✨ Ce que fait Naya
- 📱 Le portail client
- 💚 Tarif bêta
- ❓ Questions fréquentes
- 💬 Elles utilisent Naya
- 🔒 Sécurité & RGPD
- 👋 Qui suis-je ?
- 🚀 Prochaines étapes

### Étape 3 — Publier en page publique

1. Ouvre la page principale **"Naya Demo"**
2. Clique sur **"Partager"** (bouton en haut à droite)
3. Active **"Publier sur le web"**
4. Coche **"Permettre les sous-pages"** pour que toutes les pages soient accessibles
5. Copie l'URL générée → c'est ton lien de démo

**Pour une URL personnalisée :**
Notion propose `ton-nom.notion.site` via les paramètres de site Notion (fonctionnalité payante Notion).
Tu peux viser `naya-demo.notion.site` si disponible.

---

## Checklist des placeholders à remplacer

Avant de partager le lien, remplace chaque placeholder ci-dessous dans les pages correspondantes.

- [ ] `[CALENDLY_LINK]` → ton URL Calendly (pages : Accueil, Tarif bêta, Prochaines étapes)
- [ ] `[LOOM_LINK]` → ton URL Loom de démo (page : Démo vidéo)
- [ ] `[SCREENSHOT_1]` à `[SCREENSHOT_5]` → captures d'écran du portail client (page : Portail client)
- [ ] `[DATE]` → date de fin de la période bêta, ex. "1er juillet 2026" (page : Tarif bêta)
- [ ] `[PHOTOS_TEMOIGNAGES]` → photos des bêta-testeuses (page : Témoignages)
- [ ] `[PLACES_RESTANTES]` → nombre de places restantes sur 10 (page : Tarif bêta)
- [ ] Photo perso d'Oumar (page : Qui suis-je ?)

---

## Conseils de mise en forme rapide dans Notion

- Ajoute une **image de couverture** sur la page Accueil (photo nature, tons verts)
- Utilise des **blocs callout** (💡 ou 🎁) pour mettre en valeur les CTA
- Le bloc `/embed` dans Notion permet d'intégrer directement une vidéo Loom
